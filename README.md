# SignLanguageToText

## HOW TO RUN
1. **Clone the repository:**

   ```bash
   git clone https://github.com/Anuj3124/SignLanguageToText.git
   cd SignLanguageToText
   ```

2. **Install the dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**

   ```bash
   python app.py
   ```
